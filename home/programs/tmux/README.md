# Tmux Keymaps

Prefix: `C-Space`

## Navigation

| Key | Action |
|---|---|
| `C-h` | Move to left pane (works across nvim/tmux) |
| `C-j` | Move to pane below |
| `C-k` | Move to pane above |
| `C-l` | Move to right pane |
| `prefix + Tab` | Next window |
| `prefix + Shift-Tab` | Previous window |
| `prefix + Space` | Toggle last window |
| `prefix + 1-9` | Jump to window by number |

## Splits & Windows

| Key | Action |
|---|---|
| `prefix + v` | Vertical split |
| `prefix + s` | Horizontal split |
| `prefix + c` | New window |
| `prefix + x` | Close pane |

## Resize

| Key | Action |
|---|---|
| `prefix + H` | Resize left |
| `prefix + J` | Resize down |
| `prefix + K` | Resize up |
| `prefix + L` | Resize right |

## Copy Mode

Enter with `prefix + [`

| Key | Action |
|---|---|
| `v` | Begin selection |
| `C-v` | Toggle block selection |
| `y` | Yank selection |
| `Escape` | Cancel |

## Misc

| Key | Action |
|---|---|
| `prefix + r` | Reload config |
